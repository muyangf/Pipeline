# Pipeline OS 3.0

Pipeline OS is a **pipeline operating system** designed for **local/offline LLM development workflows**.  
It provides a structured, auditable, and reproducible development process across five layers:

- **Spec Layer** – Requirements, planning, task breakdown, change specs  
- **Execution Layer** – TDD, implementation, debugging, subagents  
- **Quality Layer** – Code review, refactoring, branch finishing  
- **Archive Layer** – Archiving and changelog generation  
- **Pipeline Layer** – Global orchestration and next-step engine

## Features

- Chinese-first  
- Offline-friendly  
- Stable structured outputs  
- Auditable and reproducible  
- Compatible with OpenCode, Superpowers, and OpenSpec
